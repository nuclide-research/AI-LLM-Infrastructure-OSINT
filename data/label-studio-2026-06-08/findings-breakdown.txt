METHODOLOGY LESSON
  /version returns JSON WRAPPED IN <pre>...</pre> HTML on Label Studio.
  A naive json.loads() of the response body returns "dead". The verifier
  needed a <pre>-strip pass before json.loads. Pre-fix: 0 unauth from 500.
  Post-fix: 1 unauth + 404 auth-gated + 21 FP-shape. The fix changed the
  measurement from "appears all dead" to "almost all auth-gated."

  Codifying: Insight candidate, "HTML-wrapping of API response bodies
  defeats naive JSON-only fingerprints; strip canonical wrappers before
  parsing."
