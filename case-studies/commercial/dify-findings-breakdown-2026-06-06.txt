════════════════════════════════════════════════════════════════════
FP CATALOG
════════════════════════════════════════════════════════════════════

1. CONFIG_DISC is NOT a vulnerability — /console/api/system-features
   is intentionally public. It is an information exposure that enables
   operator profiling. Severity: LOW for a single instance,
   MEDIUM at population scale (enables targeting).

2. SETUP_OPEN requires human interaction to exploit (browse to setup
   wizard). Not remotely exploitable by API alone. Still CRITICAL
   because impact is full admin control.

3. SIGNUP_OPEN with SSO enforced (F-004) requires verification of
   whether the registration path actually bypasses SSO. Rated HIGH
   pending confirmation; if SSO check runs pre-registration, downgrade
   to MEDIUM.
