# NICE 621 — Software Developer

**Source PDF:** `~/Documents/dod-cyber-pathways/621-Software-Developer-Career-Pathway.pdf`

**Role definition (NICE):** "Develops, creates, maintains, and writes/codes new (or modifies existing) computer applications, software, or specialized utility programs."

**OPM Occupational Series alignment (top 3):**
- 2210-Information Technology — 60%
- 1550-Computer Science — 26%
- 0854-Computer Engineering — 7%

**Common Work Role Pairings (top 3):**
- 632-Systems Developer — 44%
- 641-Systems Requirements Planner — 11%

## Research-program relevance

This is the role that writes the inference servers, agent platforms, and RAG pipelines NuClide assesses. Auth-on-default thesis turns on whether the 621 wired authentication into the default deployment path or left it as configurable opt-in. Functional titles include "Cloud Developer", "Full Stack Developer", "Secure Software Engineer" — directly the population shipping Weaviate clusters, MLflow servers, Ollama deployments. The Core Task list (T0022 security in requirements, T0236 threat modeling, T0118 end-of-life security) is the upstream control set whose absence produces the exposures we map.
