# NICE 612 — Security Control Assessor

**Source PDF:** `~/Documents/dod-cyber-pathways/612_Security_Control_Assessor.pdf`

**Role definition (NICE):** "Conducts independent comprehensive assessments of the management, operational, and technical security controls and control enhancements employed within or inherited by an information technology (IT) system to determine the overall effectiveness of the controls (as defined in NIST SP 800-37)."

## Research-program relevance

Closest formal NICE analog to what NuClide does, but inside-the-boundary. The 612 runs SP 800-37 assessments, evaluates control effectiveness, and writes residual-risk statements for systems they have lawful access to. NuClide's external schema-only assessments produce findings the 612 would normally generate via authenticated control testing — useful for framing disclosure reports in language an internal assessor recognizes ("residual risk", "POA&M", "deviation"). Pairing rate of 20% with 541-Vulnerability Assessment Analyst reflects the inside/outside split of the same work.
